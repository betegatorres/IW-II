O que é DOM? Explique com suas palavras e a relação com HTML.

O DOM (Document Object Model) que é uma forma di javascript se comunicar com html.

O que é um nó (node)? Explique elementos, atributos e textos

Um nó é qualquer item dentro do DOM. Tudo na página é um nó.

Explique os métodos: getElementById(), querySelector(), querySelectorAll().

getElementById() - é uma função que seleciona um elemento ou um nó pelo ID

querySelector() - Busca o primeiro elemento que são de um seletor

querySelectorAll() - Busca todos os elementos que são de um seletor e os coloca em uma lista.

Diferença entre innerHTML e innerText

O innerHTML serve para alterar o HTML dentro de um elemento

O innerText serve para alterar o texto dentro de um elemento, não interpretando HTML

O que são eventos? Exemplos: onclick, onchange, onmouseover

Eventos ativam alguma função quando ações acontecem.

onclick - acontece quando um elemento é clicado

onchange - acontece quando um elemento sofre uma alteração

onmouseover - acontece quando o mouse passa por cima
